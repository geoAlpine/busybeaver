# M4：逆問題（非Haar ＋ 整数 seed ⇒ 禁 address）— 会議資料（2026-06-27）
B_k 実験で「中央桁は自律（両端から decoupled）」が判明し、inter-scale の橋は**チャネル不在で閉じた**。残る主線は
M4＝逆問題。本資料はそれを会議で詰めるための材料。**中央の自律性を、邪魔でなく制約として逆手に取る**のが鍵。
誤証明ゼロ。

---
## 0. なぜ M4 が主線に昇格したか
- 順問題（seed 8 が散らばることを直接示す）：全ルート閉（5方向＋部分空間＋中央橋 B_k）。
- B_k≈0：中央は両端から独立 ⇒ 既知の端から中央を制御する道具は**どれも届かない**。
- ⇒ 残るのは「**中央の自律性を制約として使う逆問題**」と、Antihydra 固有の**整数性**。一般の rank-1 では不変測度が
  非可算 simplex（剛性なし）だが、**整数 seed** は特別——ここに賭ける。

## 1. M4 の標的
> **非Haar empirical measure を持つ軌道の seed は、2-adic address に「可視な禁構造」を持つ。
> そして整数 seed 8 はその構造を持てない。** ⇒ seed 8 は Haar-generic ⇒ Antihydra 非停止。
これは rank-1 測度剛性の「自家製・整数版」を作る方向。順問題が全閉した今、唯一 Antihydra 固有の突破口。

## 2. 中央の自律性を「制約」として使う（B_k≈0 の逆手）
B_k≈0（中央は両端から独立）は順問題には壁だが、逆問題には**強い制約**：
- 非Haar の偏りは、両端（foothold・×3 エンジン）から駆動され**ない**（decoupled）。だから偏りは**中央自身に
  autonomous に閉じ込められる**。
- ⇒ 非Haar 軌道は「**中央桁が自分で持続的に偏る**」しかない。これは非常に強い条件：中央の偏りを外部（端）の
  ありふれた構造で説明できないので、seed 自身の 2-adic 桁に**直接**現れねばならない。
- **狙い**：この「autonomous な中央偏り」が、整数の 2-adic 展開と**両立しない**ことを示す。

## 3. 整数性の梃子（seed 8 が一般 2-adic 点と違う点）
| 一般 2-adic 点（exceptional 可能） | 整数 seed 8 |
|---|---|
| 2-adic 桁が任意（非可算 Cantor 例外集合に乗れる） | 桁が 8·3ⁿ/2ⁿ で**3-adic・実・2-adic が連動** |
| 高さ非有界の自由な address | 各 n で c_n は**1つの整数**＝3つの完備化が同一の有理数の像 |
| 周期点・特異前像に乗れる | **既証**：整数 seed は周期部分（2-adic repelling, Lemma1）と特異前像（Haar-null 有理, Lemma2）を回避 |
- 既に Lemma1/2 で例外集合の「飼い慣らされた」部分（周期・特異前像）を整数性で排除済み。**残るのは正エントロピー
  Cantor 部分**。M4 はそれを整数性で攻める。
- 整数の決定的特徴：c_n は**真の整数**で、その中央桁は 8·3ⁿ の中央2進桁＝3ⁿ の桁構造に直結。3ⁿ の桁は
  **完全複雑度**（持続的偏りの構造を持たない、と経験的）。「中央が自律的に持続偏り」＝「3ⁿ の中央桁が持続偏り」
  ＝ 3ⁿ の桁の禁構造。

## 4. 候補定義（「禁 address 構造」を何にするか）
### (P1) ★中央の autonomous 偏りの不変量
- 定義：非Haar ν に対し、中央桁の「自律偏り」`a(ν) = (中央桁の ν-平均) − 1/2 ≠ 0`。B_k≈0 より、これは端で
  説明されない純粋な中央の量。
- 禁構造：`a(ν)≠0` が整数 seed の 3ⁿ 桁構造（完全複雑度）と両立するか。funnel 注意（3ⁿ 桁＝Mahler core）だが、
  **逆向き**（偏りの存在から構造を導く）なので順問題と違う handle になりうる。

### (P2) 3-adic / 整数性の代数的制約
- 定義：整数 c_n の存在は `2ⁿ c_n = 3ⁿ·8 − T_n`（既証恒等式、T_n は parity carry）。非Haar ⇒ parity 列 r_i が
  偏る ⇒ T_n の代数的構造に制約。整数性（c_n ∈ ℤ ⇒ T_n ≡ 3ⁿ·8 mod 2ⁿ）が偏った r_i を許すか。
- 禁構造：偏った parity が整数性の合同条件と両立しない range を探す。これは**純代数**（測度論でなく）で、
  Antihydra 固有（valuation budget・dual carry を使う）。

### (P3) 逆 shrinking-target（自律偏り ⇒ 深 cylinder 異常滞在 ⇒ 整数で禁）
- 定義：中央の自律偏り ⇒ 軌道が深 cylinder に異常滞在（even-density→0 の geometric-run シナリオ、既出）。
  整数 seed では run 長 ≤ 0.585n（既証 bit-length bound）。
- 禁構造：autonomous 偏りが要求する deep-cylinder 滞在頻度が、整数の bit-length 制約と両立するか（既出の
  「growth は marginal に even-density→0 を許す」を、autonomy 制約で締められるか）。

## 5. 正直な現実認識
- M4 も最終的には 3ⁿ の桁構造（Mahler core）に触れうる。だが**逆向き**（偏りの存在 → 構造の禁止）かつ
  **整数性**（Antihydra 固有、一般 rank-1 にない）を使う点で、順問題の全ルートと**質的に別**。
- 最大の希望は (P2)：純代数（整数性の合同条件 ＋ valuation budget ＋ dual carry）で、測度論の壁（rank-1 剛性
  なし）を**迂回**する。測度でなく**整数の算術**で攻めるのが、唯一 Antihydra 固有の道。
- 最大のリスク：rank-1 は例外測度が非可算なので、「非Haar ⇒ 可視構造」が一般には偽。整数性だけが頼り。整数性
  が effective rank-2 や剛性の代わりを**本当に**果たすかは未知。

## 6. 会議で決めること
1. **(P1)(P2)(P3) のどれを主線にするか？** 推奨は **(P2) 純代数**（整数性 ＋ valuation budget ＋ dual carry で、
   偏った parity が整数性合同と両立しない range を探す）——測度論の壁を迂回する唯一の道。
2. **「禁 address 構造」の最小定義を1つ選ぶ**：B_k≈0（中央自律）を制約として、非Haar が中央桁に強制する量を、
   整数性が排除できる形に。
3. **最初の実験**：人工的に偏った parity 列（半内生）を整数性合同 `T_n ≡ 3ⁿ·8 mod 2ⁿ` に通し、どの偏りが
   整数性と**両立しない**かを測る（順問題でなく「禁止域」の探索）。
4. **toy cryptid で先に M4 を1本証明**：o17（odometer 型）や 8/3 型など、整数性の制約が強い弱い cryptid で
   「非Haar 整数 seed ⇒ 矛盾」を実証してから Antihydra へ。

> 会議の目的：**M4 の「禁 address 構造」の最小定義を1つ選び、整数性で攻める純代数路 (P2) か逆 shrinking-target
> (P3) かを決める**こと。中央の自律性（B_k≈0）は M4 の**制約・武器**——「偏りは中央に閉じ込められる」を使い、
> 整数性（seed 8 の特別さ）で禁止する。順問題が全閉した今、これが唯一 Antihydra 固有で残る方向。
