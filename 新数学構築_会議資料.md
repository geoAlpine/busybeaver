## 5. 新数学が越えるべき正確な障害（4つの顔、同じ谷）
1. **自己参照の循環**：低桁を作る「繰り上がり」は軌道**自身の parity 出力**で駆動。新鮮ビットは**経験的に存在**
   （条件付き次ビットは一様、偏差≤0.03）が、自分の出力の新鮮さを**循環せず**証明できない。skew product で
   cocycle が系の自己出力のとき「cocycle の新鮮さ」＝「系の equidistribution」（定義的に同値）。
2. **収縮なき繰り込み固定点**：run-start cross-section の return map は同じ Gibbs-Markov（傾き (3/2)^{L+M}）。
   問題が自己相似に再生産され、**縮む方向が無い**。あらゆる cross-section・部分列が同じ難しさを再生。
3. **2変数族の対角線**：`S(N)=bit_n(3ⁿ)` は2変数族の対角。**行（固定法）は厳密に散らばり、列（固定桁）は厳密に
   周期的**。どの道具も行か列を制御するが、**対角は誰も制御しない**。
4. **measure vs arithmetic-address**：spectral gap の指数小 bad set は measure の言葉。「種が外れる」は arithmetic
   （桁住所）の言葉。両者の橋が無い。

## 6. 新数学の候補方向（INVENTすべき対象）← 会議の核心
既存の道具は全て閉じた（§3）。だから以下は「既存定理の適用」でなく「**新しい枠組みの発明**」。最も有望な順：

### (a) Diophantine 駆動の繰り込み（self-referential equidistribution の有効理論）
- **発明する対象**：スケール 2ᵏ→2ᵏ⁺¹ の「新鮮ビット」を、軌道自身でなく**外部の単調減少量**（`log₂3` の
  無理数度 F、または β展開の lap 数）から供給する繰り込み理論。これがあれば van der Corput 型の登りが
  **非循環**になる（障害1を破る）。
- **なぜ新しいか**：既存の繰り込み（Host の正規性は a.e.、van der Corput は外部の次数減少を消費）は
  自己参照入力を扱えない。`log₂3` を「per-scale の燃料」にする枠組みは無い。
- **最初の一歩**：F（連分数 F）が k 番目のスケールに供給する「情報量」を定量化し、それが障害2の繰り込みの
  expand を相殺できるか（玩具モデルで）。

### (b) rank-1 amenable-hyperbolic 軌道の derandomization 原理
- **発明する対象**：QUE が Hecke 対称性で a.e.→every を達成するのに倣い、**`log₂3` の Diophantine 構造が
  「余分な対称性」の役を果たす** derandomization 原理。spectral gap（A、Haar レベル）を指定軌道に降ろす。
- **なぜ新しいか**：文献調査で「rank-1・正エントロピー・computable」に届く derandomization 原理は皆無と確定。
- **最初の一歩**：bad set（A の大偏差）を **cylinder の合併として arithmetic に記述**し、種が F の制御下で
  外れる条件を（障害4の橋を）定式化。

### (c) 窓付き／局所 S-unit 理論（部分空間定理の密度版）
- **発明する対象**：部分空間定理を「変数の個数の有限性定理」から「**高さに線形な密度定理**」へ格上げ。
  具体的標的：`s₂(3ⁿ mod 2^{n+εn}) ≫ εn`（窓内の非ゼロ桁が正の割合）。
- **なぜ新しいか**：log 障壁は ESS 定理の構造（解の個数 ∝ exp(poly(変数数))、高さ非依存）に固有。これを
  破るのは Diophantine 近似の breakthrough そのもの。だが**欠けている補題が precise に位置づく**。
- **最初の一歩**：窓を「動く」ことの利点（連続する n で 3ⁿ→3ⁿ⁺¹ は ×3、既知構造）を S-unit 側に持ち込めるか。

### (d) endogenous cocycle のカップリング理論
- **発明する対象**：skew product で cocycle が系の自己出力のときの equidistribution 理論。既知の ×3 mixing
  エンジン（C）に parity を結ぶ。Furstenberg/Host-Kra は外生 cocycle 専用なので、**自己生成 cocycle 用の
  新しい構造定理**が要る。
- **最初の一歩**：Wiener-Wintner / return-times の枠で「[証明済み ×3 mixing] × [補題1つ：parity が good
  universal weight]」へ**モジュール化**（Gowers ノルム言語、Tao の Syracuse 条件付き定式化が雛形）。

### (e) 玩具モデル先行（理論を簡単な版で組んでから本番へ）
- 上記 (a)-(d) のどれを選んでも、まず**自己feeding を線形/有限記憶で近似した玩具**で機構を組み、その後に
  本物（完全複雑度）へ。これは「新数学」を作る標準的な進め方。
