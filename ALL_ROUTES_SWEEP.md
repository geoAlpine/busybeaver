# 全ルート総攻撃の結果（2026-06-27）— 考えうる5ルートを並列で本気で攻めた
5つの研究エージェントで、橋を架ける全ルートを並列攻撃した結果の統合。結論：**全ルートが同じ壁
（= 単一軌道 equidistribution = マーラー 3/2）に funnel する。だが各ルートは別々の正確な構造的理由
で funnel し、2つの本物の arithmetic な隙間・名前付き未解決予想との一致・ノートの訂正が出た。**
引用は検証済/未検証を明記、捏造ゼロ。

## 大きな発見1：これは名前付きの現行未解決予想だった（ルートD）
我々の「下位 2-進桁の equidistribution」は、**2025年の Andrieu–Eliahou–Vivion「有理基数体系の正規性予想」
（arXiv:2510.11723, Conjecture 1.2）を基数 3/2 に特殊化したもの**と**完全に一致**。
- 軌道の parity/低桁列の minimal word が {0,1} 上で**正規**であろう、という予想（＝全 k で長さ k の各ブロックが
  密度 2⁻ᵏ ＝ 全 mod 2ᵏ で equidistribution）。
- AEV は「**0 が各 minimal word に少なくとも1回現れる**」（我々の even-density よりはるかに弱い）すら
  **未解決**で、それですら Akiyama の三進展開予想を解くのに十分、と述べている。
- ⇒ **学界の最前線は我々の even-density よりさらに手前にある。** 我々の問題は認識された open conjecture そのもの。

## 大きな発見2：2つの本物の arithmetic な隙間（ルートE）
両方とも部分空間定理（Schlickewei の p-進版）由来で、measure でも Fourier でもない＝**全ての壁と直交**：
1. **3ⁿ の桁の連の長さ**（Lagarias, arXiv:2501.00850）：**無条件で、3ⁿ の2進展開の等しいビットの最長連は o(n)**。
   - 一見の partial：Antihydra は「長さ ≥ εn の一定 parity の連（streak）」では停止できない。
   - **だが検証の結果 marginal（過度に主張しない）**：(a) 我々は既に「奇数連 = v₂(c−1) ≤ 0.585n」を自前で
     持ち、「1つの巨大 streak での停止」モードは既にほぼカバー済み。(b) 実際の最長連は N=2e5 で**たった20**
     （≈O(log n)）＝両 bound より遥かに小さい。(c) **連 ≠ 頻度**：停止は累積密度の crossing で起き、単一の連
     では起きない——どちらの bound も停止を**防がない**。(d) 部分空間 o(n) の Antihydra 軌道への接続は要 PDF 確認。
   - **結論**：本物の arithmetic な隙間で全壁と直交するのは事実だが、**現状では breakthrough でなく marginal**。
     価値は「最も直交した攻撃面が部分空間定理にある」と特定したこと。窓付き S-unit 下界（下記）が真の鍵。
2. **ビット格子の 2-進自己相似性**（Lagarias 0902.3257）：n→2n（n=2ᵏ に沿って）で厳密な arithmetic 自己相似。
   部分列 n=2ᵏ に沿った頻度は**まだ誰も計算し尽くしていない**らしい＝具体的な数値実験の候補。

## ノートの訂正（エージェントが load-bearing な誤りを指摘）
1. **sum-product の失敗理由が間違い**（`mahler_equidistribution_attack.md` §4）。「部分群 \|H\|≈log q が小さすぎる」
   は**不正確**。実は H=⟨3⟩ mod 2ᵏ は位数 2ᵏ⁻²≈q/4 で**完全な単位群**（最大）。BGK/Bourgain-Chang の仮定は
   **過剰に満たされ**、部分群上の和は厳密な Ramanujan 和の打ち消し（=0）。本当の壁は「**対象が部分群の和でなく、
   対角線上の1点（n番目の冪のn番目のビット）**」＝次元の不一致。
2. **Gowers ノルムは van der Corput と同じ理由で閉じる**（全次数で差分作用素が (3/2)ⁿ を不動点に持つ）。
   ＝「van der Corput closed」の自然な拡張として記録すべき。

## 各隙間を開くには何が真でなければならないか（次に探すべきもの）
- **ルートE-1（部分空間）**：窓付き S-unit 下界 `s₂(3ⁿ mod 2^{n+εn}) ≫ εn`（Stewart の大域版 ≫ log n/log log n を
  線形に強化）。それ自体が問題と同格の breakthrough だが、**欠けている補題が Diophantine 近似の中に正確に位置づく**。
- **ルートD（bootstrap 非循環化）**：スケール 2ᵏ→2ᵏ⁺¹ で**厳密に減る外部の単調量**（超越度/無理数度、または
  β展開の lap 数）。あれば van der Corput 型の登りが非循環になり、新鮮ビットが**外部から**供給される。
- **ルートD（Wiener-Wintner）**：問題を [証明済みの ×3 mixing] × [補題1つ：parity が good universal weight] に
  **モジュール化**（Gowers ノルム言語で）。Tao の Syracuse の条件付き定式化が雛形＝正直な天井は「条件付き定理」。

## 最後の隙間も部分空間定理で攻めた→閉じている（2026-06-27、log 障壁）
唯一の有望な隙間「窓付き S-unit 下界 `s₂(3ⁿ mod 2^{n+εn}) ≫ εn`」を部分空間定理で正面から攻撃。
**結果：閉じている。理由は構造的（log 障壁）。**
- **対象の正確化**：偶数密度 ＝ 3ⁿ の動く対角位置 n−3 のビットの 0 の密度（`c_n mod 2 = bit_{n−3}(3ⁿ)`）。
  欲しいのは「3ⁿ の窓内の非ゼロビット数 ≫ εn」（＝正の桁密度）。
- **真実は線形**：3ⁿ は約 0.79n 個（密度 ~0.5）の非ゼロビットを持ち、対角窓内も ~0.5（＝まさに偶数密度）。
- **部分空間定理が与えるのは log だけ**：`3ⁿ = Σ 2^{aᵢ}`（k＝非ゼロ桁数）は S-unit 方程式。定量的部分空間定理
  （Evertse-Schlickewei-Schmidt）は非退化解の**個数**を**項数 k の関数 exp(poly(k)) で**bound する——**高さに
  依らない**。特定の 3ⁿ（高さ ~n）に合わせると `k ≫ log(高さ)/log log = log n/log log n`（Stewart）しか出ない。
  `k ≫ n`（線形）を出すには高さに線形な bound が要るが、部分空間定理は**有限性/個数の定理であって密度の定理
  でない**ので、それは原理的に出せない。**log 障壁は構造的で、緩いのでない。**
- **窓付きはむしろ悪化**：窓は全体よりビットが少ないので、窓付き下界 ≤ 全体の Stewart bound = O(log n) ≪ εn。
- **結論**：部分空間定理ルートは**閉じている**。log → linear の gap がマーラー問題そのもの。最も直交した arithmetic
  な攻撃面も、その固有の log 障壁で届かない。

## 全体の正直な結論（最終）
**考えうる全ルート（5方向＋最後の部分空間の隙間）を攻め尽くした。全て閉じている。各々別の正確な構造的理由で。**
これは「全部試して無駄だった」ではなく「**全部試して、どこにも近道がないことを各方向の precise な理由とともに
証明した**」。完全証明は genuinely 新しい数学を要する——既存のどの道具（最も直交した部分空間定理ですら）も届かない、
ということが exhaustive に確定した。
