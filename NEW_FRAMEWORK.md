# A self-consistent (mean-field) transfer operator for self-generated cocycles — framework design
Design of the new mathematics the GM skeleton's line (5) requires: a transfer-operator framework in which
the scenery is GENERATED BY the orbit (closed loop), so the operator depends on the measure it acts on.
This is where the Antihydra genericity problem naturally lives. Honest throughout: the design is genuinely
new (the object does not exist in standard form); the single new lemma it isolates is the Mahler core in
modern transfer-operator language, not a dissolution of it. [DESIGN] / [PROVEN] / [OPEN] labelled.

## 1. Why the standard transfer operator is the wrong object here
The Antihydra map `T(c)=(3c − (c mod 2))/2` is a continuous map of `ℤ₂` (verified: `3c−r ≡ 0 mod 2`).
- **[PROVEN]** Its Perron–Frobenius operator `L` w.r.t. Haar is **one-step exact** (2nd eigenvalue ≈0,
  uniform stationary; `transfer_carry.py`/`gm_skeleton.py`). So Haar-a.e. point equidistributes.
- **[PROVEN/OBSERVED]** The orbit of `8` is an INTEGER orbit; integers are dense but **Haar-null** in `ℤ₂`,
  so the a.e. statement does not reach it. (This is line (5).)
The standard operator `L` is *linear* and uses a *fixed* reference measure (Haar). It cannot see a single
Haar-null orbit. The missing structure: along the real orbit the "incoming high bit" (the scenery driving
each step) is **not an external fair coin** — it is the orbit's own higher digit. The operator that governs
the orbit must therefore be built from the orbit's OWN statistics.

## 2. The new object — a measure-dependent (self-consistent) transfer operator
[DESIGN] For a probability measure `ν` on `ℤ₂`, define the **`ν`-conditioned step operator** `L_ν` by
replacing the incoming high bit's law with its `ν`-conditional given the current low state:
> `L_ν(s → s') = Σ_{b∈{0,1}} P_ν(\text{high bit}=b \mid \text{low state}=s)\;[\,\text{step}(s,b)=s'\,]`.
- If `ν = Haar`, the high bit is independent and fair: `L_Haar = L` (the one-step-exact operator), and
  **Haar is a fixed point**: `Haar = L_Haar · Haar`. **[PROVEN]**
- For the real orbit, the scenery is self-generated, so the governing measure must satisfy the
  **self-consistency (mean-field) equation**
> **(SC)** `ν = L_ν · ν`.
This is a *nonlinear* fixed-point equation (the operator depends on its own invariant measure) — the
defining object of the framework, the 2-adic analogue of a mean-field / McKean–Vlasov stationary equation.
The orbit's empirical measure, IF it converges, must solve (SC).

## 3. The single new lemma the framework isolates
[OPEN = Mahler, restated] Write `ν = Haar + εδ` and linearize (SC) at Haar:
> `δ = (L + F)·δ`, where `L` is the (one-step-exact) annealed operator and **`F` is the self-generation
> feedback** — the response of the incoming-bit conditional law to a perturbation of `ν`.
Because `L` is one-step exact (spectral radius ≈0 on the mean-zero complement), the stability of Haar is
governed by **`F` alone**:
> **NEW LEMMA TO PROVE (sub-criticality).** The self-generation feedback is a contraction on the mean-zero
> complement: `‖F‖ < 1` (equivalently, the linearized self-consistency map `L+F` has spectral radius `< 1`).
If true, Haar is the **unique stable** solution of (SC) near Haar, and (with a single-realization tracking
/ propagation-of-chaos step, §5) the orbit's empirical measure is Haar ⇒ genericity ⇒ the complete proof.
- **Honest identification:** `F` is exactly the response of the **moving-middle digit / carry bit** to the

  **sub-criticality / spectral-radius** condition on an explicit feedback operator, the modern form of the

## 4. What the framework BUYS (why it is the right design, not just relabelling)
- It converts "genericity of one Haar-null orbit" (which no a.e. theorem can touch) into a **fixed-point /
  stability** question about a measure — a question that fixed-point theory (Schauder/Banach) and

- It identifies the **one scalar/operator to estimate** (`F`), turning a vague "new math" into a concrete

- It connects to an existing mature theory — **mean-field / McKean–Vlasov stationary equations and

  Keller–Zweimüller-type coupled-map operators) — so the build can borrow that machinery rather than start

## 5. The remaining gap after sub-criticality (named, for honesty)
Even granting `‖F‖<1` (unique stable Haar fixed point of (SC)), one needs the **single real orbit to track
the mean field** — that its empirical measure converges to the (SC)-solution, not merely that the solution

for the fixed point, quantitative for the tracking).
