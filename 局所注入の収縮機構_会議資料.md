# 局所注入 Inj(k) の収縮機構をどう定義するか — 会議資料（2026-06-27）
新数学・第一歩で枠組み(d)（endogeneity defect）を構築し、再帰 `Def(k+1) ≤ ρ(k)Def(k) + Inj(k)`、ρ<1 を確認、
壁を局所注入 `Inj(k)~0` に局在化した。本資料は**次の核心**「Inj(k) を収縮/制御する機構をどう定義するか」を会議で
詰めるための材料。誤証明ゼロ。

---
## 0. 出発点（確定事項）
- **ρ<1（繰り越し defect の収縮）は存在**＝×3 mixing エンジン／spectral gap（材料 A・C）。
- **壁は Inj(k) に局在**：`Inj(k)` ＝ 低位 scale k での「次ビットの不均衡」＝ c_n の低位ビット ＝ **3ⁿ の動く中央桁
  （moving-middle-digit）＝ マーラー core**。経験的に i.i.d.-ゼロ（control 確認）だが証明が壁。
- **必要なのは**：ρ（繰り越しの収縮）とは別に、**Inj 自身を制御する σ<1 の機構**。これが無いと再帰は
  `Def_∞ ≤ Inj_sup/(1−ρ)` で Inj に詰まる。

---
## 1. なぜ「測定」では決まらないか（先に明示）
実軌道の Inj(k) は i.i.d. noise floor（control 一致）。だから Inj(k) vs Inj(k−1) も noise vs noise で、
**σ は実軌道の測定からは出ない**（Def と同じ）。σ は**作用素レベル**（注入された注入欠陥がどう伝播するか）で
定義・解析するしかない。⇒ 会議の仕事は「**収縮を測る作用素/不変量を定義する**」こと。

---
## 1.5 核心の幾何：2つの foothold の間の「中央ギャップ」（重要・自己訂正）
- archimedean foothold は **どの無理数でも Θ(log N)**（N 個の点の discrepancy ~1/N が限界。連分数が周期的でも
  定数が変わるだけで Θ(log N) のまま——「周期的連分数＝全スケール」は**誤り**）。上位 ≈0.85 log₂N 桁のみ。
- ×3 エンジンは **下位の固定 scale の桁**を制御（3ⁿ mod 2ᵏ は固定 k で完全周期）。
- ⇒ c_n のビット列は【上位 log N：foothold】…【中央 Θ(n)：制御なし】…【下位 fixed：×3 エンジン】。
  **Inj(k) はこの中央ギャップに住む**。そして「2-進 foothold」を作っても**同じく log N** で中央に届かない。
- **結論**：必要な新機構は「もう一つの foothold」では**ない**。**Θ(n) 幅の中央ギャップを橋渡しする**もの。
  既存の道具（2つの foothold＝両端 log N、繰り込み＝収縮なし、部分空間＝log 障壁）は**どれも中央を渡れない**。

## 2. 候補：Inj(k) 収縮機構の定義（4案、すべて「中央を渡る」ことが課題）
### (M3) ★中央ギャップを渡る propagation 機構（foothold ではなく「橋」）
- **発明する対象**：上位 foothold の制御を、繰り込みを通じて**中央へ Θ(n) 桁伝播**させる機構。繰り込み自体は
  収縮しない（固定点）ので、伝播の各段で**外部の単調量**を消費して非自明に降ろす必要。
- **なぜ核心か**：Inj は中央に住み、両端の foothold/エンジンは届かない。橋＝両端の既知制御を中央でつなぐもの。
- **最小の定義**：上位 foothold の「残差」（深さ k の制御誤差 D*_N）が、繰り込み1段で中央側へどう変換されるかの
  作用素 `B_k`。`B_k` が**収縮的（ノルム<1）+ 外部単調量で誤差が summable**なら、log N の foothold が中央へ
  伸びる。これが σ<1 の正体。
- **正直な注**：繰り込み(return map)は傾き (3/2)^{L+M} で**expand**（収縮の逆）。だから `B_k` を収縮させるには
  expand を相殺する外部入力が要る——それが log₂3 の Diophantine 情報を**伝播の燃料**として使う所（ルート a の
  正しい使い方＝「端で注入」でなく「各伝播段で消費」）。まだ存在しない機構。標的は precise（中央橋）。

### (M1) メタ再帰：注入の注入（second-order defect）
- **定義**：`Inj` 自身の defect `Def⁽²⁾_k`、メタ再帰 `Inj(k) ≤ σ·Inj(k−1) + meta-error`。σ<1 なら Inj→0。
- **正直な注**：これは two-scale bootstrap の一段上＝NEW_ENGINE で「self-closing でない（各スケールに新鮮ビット
  が要る）」と判明済み。Diophantine 入力 (M3) が無いと σ=1 で詰まる。⇒ (M3) の上に乗せる枠。

### (M2) カップリングの相互相関を機構にする
- **定義**：parity 入力 r_i と ×3 carry エンジンのビットの相互相関 `X_k(N)`。機構＝「X_k→0 なら fresh bit は
  fresh」。
- **正直な注**：X_k→0 は経験的に成立（~0.5=decorrelated）だが、その証明＝parity が engine に対し「十分ランダム」
  ＝equidistribution。(M3) の燃料が無いと循環。⇒ (M3) を X_k の証明に注入する形。

### (M4) 逆問題／自家製 rank-1 剛性（ユーザー案2）
- **定義**：非等分布 seed が持つ「禁じられた 2-進 address 構造」を分類し、整数 seed 8 がそれを持てないことを示す。
- **正直な注**：rank-1 は不変測度が非可算 simplex（剛性なし）なので「禁じられた構造」の pin が難しい。だが
  **整数性**（seed 8 の特別さ＝有界高さでない／2-adic repelling）が効く余地はある。独立の側道。

---
## 3. 推奨する定義の順序（(d)の上に「中央橋 B_k」を乗せる）
1. **(M3) 中央橋 B_k を最小形で定義**：renewal return map の transfer 作用素を foothold 残差の部分空間に制限した
   伝播作用素。その収縮率 σ が目標。
2. **(M1) メタ再帰の器に B_k を入れる**：`Inj(k) ≤ σ·Inj(k−1) + (伝播段の log₂3 燃料)`。σ<1 が出るか。
3. **(M2) で循環を閉じる**：X_k→0（parity と engine の decorrelation）の証明に B_k と log₂3 燃料を使う。

---
## 4. toy model（機構を学ぶ実験場、正しい版）
- ★**半内生モデル**（これが正しい toy）：parity 入力を λ-外生化（λ-割合をランダムに）。λ=1 で i.i.d.（自明に
  等分布）、λ=0 で実軌道（壁）。**どの λ で中央橋 `B_k` が収縮し cancellation が出るか**を測る（外生→半内生→
  内生）。臨界 λ のすぐ上で効く機構を見れば「必要な燃料の量と形」が分かる。
- **注（自己訂正）**：「周期的連分数の二次無理数で foothold が全スケール」は**誤り**——foothold は任意の無理数で
  Θ(log N)。だから二次無理数 toy では中央ギャップは消えない。toy で操作すべきは**外生性 λ**（橋の燃料の代理）で
  あって基数の Diophantine 性ではない。

---
## 5. 会議で決めること
1. **中央橋 `B_k`（伝播作用素）を、どう定義するか？** ＝上位 foothold の残差 D*_N が繰り込み1段で中央側へ
   どう写るか。最小定義：renewal return map の transfer 作用素の「foothold 部分空間への制限」、その収縮率を σ に。
2. **半内生 toy（λ-外生化）で臨界 λ と機構を測る。** どの外生量で B_k が収縮するか＝必要な燃料の形。
3. **log₂3 の Diophantine 情報を「端で注入」でなく「各伝播段で消費」する形に定式化できるか**（ルート a の正用法）。
4. **(M4) 逆問題（非等分布 seed の禁構造＋seed 8 の整数性）を並走させるか**（独立の側道）。
5. **正直な現実認識**：4案とも中央 Θ(n) ギャップを渡る課題に帰着。両端 foothold は届かず繰り込みは収縮しない。
   だが**(d) の器＋「中央橋 B_k」の定義**は、証明の標的を「両端の既知制御を中央で σ<1 でつなぐ伝播作用素」と
   いう **precise な対象**に縮める——これが新数学の次の最小単位。B_k の収縮（σ<1）が出るかが、ユーザーの
   「step 2 が決める」の本当の中身。

> 会議の目的：**中央橋 `B_k`（foothold 残差 → 中央への伝播作用素）の最小定義を1つ選ぶ**こと。その σ<1 が
> 収縮機構。半内生 toy で σ<1 を実証し、log₂3 を伝播段の燃料として注げれば Antihydra に戻る道が見える。
> 「もう一つの foothold」ではなく「**中央を渡る橋**」——ここが核心。
